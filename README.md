## 🧬 Projetos em Destaque

### 📦 Afilibot | Engenharia de Software e Infraestrutura Cloud
> Plataforma Fullstack de alta complexidade com infraestrutura 100% automatizada e orientada a microsserviços.

- **Arquitetura & Backend:** Desenvolvida com Clean Architecture e Repository Pattern. Backend construído em **FastAPI**, com divisão clara de responsabilidades (app roles isoladas para web, workers e scrapers), garantindo resiliência e escalabilidade horizontal.
- **Banco de Dados:** Modelagem relacional rodando em **Azure PostgreSQL** (mais de 12 tabelas), com gerenciamento de relações N:N, controle de pagamentos e agendamento de filas para workers nativos.
- **DevOps & Cloud:** Aprovisionamento de infraestrutura utilizando **Terraform (IaC)** para criar Máquinas Virtuais no GCP. Conteinerização total da aplicação com **Docker** e orquestração de deploy via pipelines YAML no **GitHub Actions**.

---

### 🤖 Processamento Inteligente de Documentos | Human Power
> Pipeline de dados e IA para validação semântica de alto volume de PDFs.

- **O Desafio:** Eliminar o gargalo operacional na triagem e processamento manual de documentos corporativos (contratos, ASOs, aditivos).
- **A Solução:** Desenvolvimento de aplicação web integrando extração de texto via **OCR (Pytesseract)** com algoritmos próprios de saneamento de dados. Integração profunda com **LLMs (Google Gemini API)** para realizar a classificação e validação semântica automática.
- **DevOps & Sustentação:** Manutenção direta de ambientes **Linux** e implementação de esteiras de **CI/CD** para garantir estabilidade nas atualizações.
- **O Impacto:** Otimização operacional drástica — redução do tempo de processamento por lote de **~1 hora para ~20 minutos**, eliminando retrabalho manual e reduzindo carga nos servidores.
